What you'll learn

Practical, project-based knowledge to build and launch your full-stack
application. The bootcamp is split into two parts.




Questions? Get in touch

module 1

Intro to Coding, Computer Science, and App Development

Learn the basics of coding, how to build a website and mobile app as well as how web servers, domains and hosting works. You’ll also learn about how to build apps quickly using a variety of No Code Solutions.

What's covered?

module 2

HTML, CSS, and Web Development

Build a website from scratch with HTML & CSS, participate in a practical Coding Session and upload your website online using GitHub and Netlify

What's covered?

module 3

Javascript & Python Programming

In this module you will learn the basics of Javascript, and Python Syntax and set up the environment to start coding on your machine.

What's covered?

module 4

Data Structures and Algorithms

Learn computer science data structures and how to solve advanced algorithms

What's covered?

module 5

Backends & Databases

In this section we’ll be covering JavaScript Backends & Databases, how to use JavaScript with Node JS, Express, MongoDB, and Mongoose to create state of the art applications.

What's covered?

module 6

Cloud, APIs, and DevOps

Learn how to host, deploy and scale cloud applications

What's covered?

module 7

Advanced UI with ReactJS

This comprehensive, intensive module covers how to build a complete frontend application with React, with an additional section on how to create mobile apps using React Native.

What's covered?

module 8

Final Project Demo & Wrap Up

This module is the culmination of all of your hard work, where you’ll put everything you’ve learned into a full stack web app and deploy it online
